## Point

**Point** is a general-purpose, AI-first language for application logic. You write rules, routes, pages, pipelines, and commands in `.point`; the compiler emits JavaScript, TypeScript, and Python. Agents get structured `check-json` diagnostics — stable refs, expected fields, and repair hints — instead of pasting entire Next.js scaffolds into chat.

```bash
npm install -g @hatchingpoint/point
point check myfile.point
point box src/app.point
```

Free and open source (MIT). CI-measured benchmarks show **~80% fewer agent tokens** on real app and repair tasks vs TypeScript paste workflows.
